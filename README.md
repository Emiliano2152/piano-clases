# Teoría Musical para Pianistas – Módulos Iniciales

Este repositorio contiene material teórico organizado en módulos para estudiar piano desde una perspectiva clara y funcional. La idea es aprender **lo mínimo necesario para tocar**, sin abrumarse con términos o teoría innecesaria al inicio.

## Módulos

1. **Intervalos**  
   Concepto básico para entender distancia entre notas y cómo suenan entre sí.

2. **Escalas**  
   Construcción de escalas mayores y menores, y cómo se relacionan con la música tonal.

3. **Funciones Armónicas (versión simple)**  
   Cómo funcionan los grados dentro de una tonalidad sin entrar en tecnicismos: tónica, subdominante y dominante, explicado de manera práctica.

Cada módulo tiene:
- Explicación simple (`README.md`)
- Ejercicios (`ejercicios.md`)

Podés completar los ejercicios creando un **fork** y enviando un **pull request** con las respuestas.
