# Audioperceptiva Básica (con teoria.com)

El objetivo de este módulo es entrenar el oído para reconocer:
- intervalos,
- direcciones melódicas,
- función de las notas dentro de una tonalidad,
- sensación de reposo/tensión.

Vamos a usar la página **teoria.com**, que tiene ejercicios interactivos simples y bien hechos.

No hace falta sacar notas exactas siempre; muchas veces lo que importa es la **sensación**.
